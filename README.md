# Подкаст "Фронтенд Хвиля"

Подкаст, присвячений найважливішим новинам у світі фронтенд-розробки. Створений на основі дайджесту новин від Olexander Smolianinov, який публікується у Front-end спільноті Dou.

Автор подкасту не має жодного офіційного зв’язку з Dou і створює його виключно з власної ініціативи.

- [Front-end спільнота Dou](https://dou.ua/forums/tags/Front-end/)
- [Статті автора Olexander Smolianinov](https://dou.ua/users/dersmoll/articles/)
